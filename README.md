# Modelagem-de-Dados-em-Grafos-de-um-serviço-de-streaming.
Um projeto feito no Neo4j AURA, que simula um serviço de streaming. Nele será encontrado um script em .cypher.

Organizei o codigo .cypher em partes cada uma executando uma certa parte do projeto, como pode ser visto no codigo.
Tive dificuldade em certas partes dos códigos, em relação a deixar ele de forma 'separada', no inico eu tentei criar 
tudo de uma vez, isso dificultou as coisas pois eu não estava conseguindo fazer os relacionamentos da forma correta e
de forma que pudesse ser vista com clareza no código, nesse momento eu tive que ver como eu poderia organiar o código 
de uma forma melhor, usei IA para que ela me auxiliasse na organização do codigo.
